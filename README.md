# gnusay-rust
GNU say made using Rust

```
   ___________ 
  <I love GNU!>
   ----------- 
    \   / 
     \ / 
 .= .-_-. =.
((_/)o o(\_)) 
 `-'(. .)`-' 
  /| \_/ |\ 
 ( | GNU | ) 
 /'\_____/'\ 
 \__)   (__/
```

## Install

If you have Rust: `cargo install gnusay`

## Usage

```bash
gnusay 0.1.0
DEADBLACKCLOVER <deadblackclover@protonmail.com>
GNU say made using Rust

USAGE:
    gnusay <TEXT>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <TEXT>    Text to display
```
